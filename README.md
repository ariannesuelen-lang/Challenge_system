# Desafio de Votação - Front-end

Este projeto é a interface utilizada pelos alunos para realizar a votação de um desafio da turma.

## Objetivo

Disponibilizar uma tela simples e intuitiva para que os alunos possam enviar seus votos e interagir com o sistema de votação.

## Funcionalidades

- Exibir a tela de votação
- Permitir que o aluno escolha sua nota/voto
- Enviar os dados para o back-end
- Exibir confirmações e mensagens de erro
- Melhorar a experiência de votação da turma

## Fluxo da aplicação

1. O aluno acessa a tela de votação
2. Seleciona o desafio ou confirma os dados exibidos
3. Informa seu voto
4. O voto é enviado para o back-end
5. O sistema retorna uma mensagem de sucesso ou erro

## Tecnologias utilizadas

- Linguagem: Python
- Framework: Streamlit
- Estilização: CSS
- Consumo de API: Restful
